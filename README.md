# Todo List

A simple and interactive Todo List application built with HTML, CSS, and JavaScript. The application uses LocalStorage to save todos so that your tasks persist even after refreshing the page.



## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage with LocalStorage
